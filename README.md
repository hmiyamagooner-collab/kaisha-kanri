| ファイル | 中身 |
|---|---|

# オフィスクエスト（index.html）

昔のファミコンRPG風の見た目で、会社のいまの動きをひとめで見渡せる画面です。

画面は1画面3列構成：**左＝オフィスマップ／中央＝時計・なかまたち・できごと／右＝住人カード（Company Dashboard）**。住人カードには今後AIを配置予定です。画面幅が狭いときは住人カードが下に回り込みます。

## マップの構成

道路は **T字**。横に走る長い直線道路の海側に公園・プール・ビーチが並び、縦の短い直線道路の反対側（東側）に商談ビルがあります。

| 場所 | 中身 |
|---|---|
| 本社オフィス | 机6つ＋パソコン6台、コピー機1台、書類棚3つ |
| 社長室 | 社長デスク・書棚 |
| 会議・応接室（セット） | 左に会議テーブル、右に応接ソファ。会議も来客対応もここ |
| 公園（コンサート会場） | 長い道路ぞい・ステージつき |
| プール | 長い道路ぞい・海岸沿いにある |
| ビーチ・海 | プールのとなり。休憩の散歩コースでもある |
| デイサービスセンター | 丸テーブル・施設長デスク |
| 駐車場 | 送迎車が停まっている。ゲートから長い道路へ出る |
| 商談ビル | 短い直線道路の反対側。丹治さんが横断歩道を渡って商談に通う |

## デイサービスは午前・午後の2部制

品川さんが送迎車で3往復します。

1. **9時**：午前の部のお迎え（ウメさん・ゴロウさんが来所）
2. **12時**：午前の部のお送り＋午後の部のお迎えを1往復で（入れ替え）
3. **16時**：午後の部のお送り

車が道路を走って画面の外へ行き、お年寄りを乗せて戻ってきます。来所中の利用者さんは右の一覧にも表示されます。

- **現在時刻**：実際の時計に連動。カレンダー連携時はこちら

## ヘルプ機能

「ヘルプをよぶ」→ 手が空いている人を選ぶと、応接コーナーまで来てくれます。

## Googleカレンダー連携

1. Google Cloud Console で Google Calendar API を有効化し APIキーを発行
2. 各自のカレンダーを「一般公開して共有」に設定
3. 「カレンダー連携」ボタンからAPIキーとカレンダーIDを登録

# Gooner 統合ポータル（gooner-portal.html）

「オフィス管理（Company Dashboard）」と「GOONER TREASURY 入出金クエスト」を1つに統合したアプリです。

- **◆ オフィス管理**タブは1画面構成：左にオフィスクエストのマップ、右に住人カード（7部署・AI謁見チャット予定）が並びます。狭い画面では上下に積み重なります
- **▼ 入出金クエスト**タブで金庫室・記録・精算クエスト・商流マップ・台帳・納税タンクに切り替えられます
- 右上の「⌂ オフィスクエスト（マップ）」からドラクエ風マップも開けます
- 最後に開いていたタブを記憶します。`gooner-portal.html#treasury` で入出金側を直接開けます
- データはそれぞれ元のアプリと同じ localStorage キーに保存されるので、単体版で入れた記録もそのまま引き継がれます
- オフィス管理のAI機能（円卓会議・謁見）は元ファイルと同様に `AI_ENDPOINT` の設定が必要です（未設定でも画面は動きます）
