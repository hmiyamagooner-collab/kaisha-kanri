# PDF → CF 取込

スプシ連携は使いません。月次CF表や請求書の **PDFを貼り付け／選択** すると、表・金額・引落日／支払期日を読み取り、入出金予定と合計を自動反映します。

## できること

| 読み取り対象 | 対応 |
|-------------|------|
| 日別カレンダー表（1〜31日） | ○ |
| 引落日・支払期日・月末 | ○ |
| 項目／内容／部署／支払方法 | ○ |
| 収入・固定費などの区分 | ○ |
| 入金合計・出金合計・差引の自動計算 | ○ |

## 精度の目安

- **テキスト付きPDF**（スプシやExcelから「PDFで保存」）→ 最も正確
- **スキャン／写真のPDF** → ページ画像＋AIで読み取り（表が崩れると精度低下）

## 使い方

1. CF画面 → PDFをドロップ／クリック選択（**複数可・最大3か月分**）／Ctrl+V
2. **PDFを読み込んで反映**（同じ月は上書き、別月は積み上げ。保持は直近3か月）
3. **CF表**タブでスプシ風の日別表（項目・内容・引落日・1〜末日・合計）を確認
4. **CF予測**で「何が足りなくなるか」を項目つきで確認

## 必要環境

サーバー側に `OPENAI_API_KEY`（円卓と同じキー）が必要です。
